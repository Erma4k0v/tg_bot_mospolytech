# В новом проекте pycharm создаем проект
# Внутри проекта в консоли прописываем следующие команды:

# Клонируем репозиторий:
git clone https://github.com/Erma4k0v/tg_bot_mospolytech.git

# Открываем в текущей директории:
cd tg_bot_mospolytech

# Выбираем ветку в которой будем работать:
git checkout 1-work

# Чтобы изменения с главной ветки ветки появились в вашем коде пишем следующее:
git pull origin main

# После каких-либо изменений для сохранения кода используем следующие команды:
git add .
git commit -m "Добавил комментарий в main.py"

# Чтобы изменения на сервере на вашей ветке видели другие разработчики вводим данную команду:
git push origin 1-work

#Теперь ваши изменения видны в проекте гитхаб

# Чтобы предложить внести ваши изменения в главную ветку main, переходите в гитхаб -- Pull requests -- New pull request
# В base - main, в compare - ваша ветка.
